<h1>How to run the new maps: </h1>

<h2>Note: change the parameters accordingly as necessary for Theta* </h2>

<h3>Map 800: (Large map with complex obstacles, maze-like)</h3>

<h4>python3 run.py -m map800.txt -p astar -s 0 0 -g 798 786 --epsilon 1</h4>
<br>
<h3>Map 800_sparse: (Large map with Few,Small Obstacles)</h3>

<h4>python3 run.py -m map800_sparse.txt -p astar -s 1 1 -g 90 700 --epsilon 1</h4>
